# Sync Regression Checklist

## Pre-check

1. 일반 계정 2개 브라우저 세션 준비
2. 익명 계정 1개 세션 준비
3. `node .\scripts\security-preflight-check.js` 통과 확인

## Scenario A: 일반 계정 동기화

1. 기기 A에서 문서 수정 후 저장
2. 기기 B에서 수동 동기화 실행
3. 수정 내용이 반영되는지 확인

## Scenario B: 충돌 분기

1. 기기 A/B에서 같은 문서를 각각 수정
2. A 동기화 완료 후 B 동기화 시 충돌 감지 확인
3. 원격 유지/로컬 덮어쓰기 분기 동작 확인

## Scenario C: 암호화/복호화

1. 일반 계정 로그인 직후 암호 해제 모달 표시 확인
2. 올바른 비밀번호 입력 시 정상 로드 확인
3. 잘못된 비밀번호 입력 시 실패 메시지 확인

## Scenario D: 자동 재시도

1. 네트워크를 일시적으로 차단하고 자동 동기화 대기
2. 재시도 메시지와 시도 횟수(최대 3회) 확인
3. 네트워크 복구 후 수동 동기화 성공 확인

## Scenario E: 익명 계정

1. 익명 로그인 후 편집/저장
2. 암호화 모달이 뜨지 않는지 확인
3. 로그아웃 시 자동 탈퇴/데이터 정리 확인
