# PART 04. 달력/기록 UI

작성일: 2026-02-16

## 1) 목표

- 달성 기록 영역을 월 단위 탐색 가능하게 만들고, 달력/표 보기 전환을 단순화하는 것.

## 2) 주요 변경

### 월 이동 + 연월 헤더
- 이전/다음 달 버튼 추가
- `YYYY년 M월` 헤더 표시
- 월 변경 시 달력 + 표가 동일 월 기준으로 동기 렌더
- 관련 커밋: `53dd4e1`, `0d9ca7d`

### 보기 모드 단순화
- 기존 `달력 보기`/`표 보기` 2버튼을 단일 토글 버튼으로 통합
- 현재 모드에 따라 버튼 라벨 자동 전환
- 관련 커밋: `53dd4e1`

### 달력 그리드화
- 요일 헤더(`일~토`) 추가
- 월 시작 요일 오프셋(앞 빈칸) 반영
- 마지막 줄 정렬을 위한 뒤 빈칸까지 채워 7열 유지
- 오늘 날짜 강조 스타일 추가
- 관련 커밋: `c0e1970` (+ 직후 보강)

## 3) 영향 파일

- `app.js` (`calendarCursor`, `shiftCalendarMonth`, `renderCalendar`, `renderCalendarTable`)
- `index.html` (월 이동 버튼/토글/연월 헤더)
- `styles.css` (요일 헤더/오늘 강조/빈칸 스타일)

## 4) 현재 동작 기준

- 월 이동 시 달력/표 데이터가 함께 바뀜
- 오늘 날짜는 현재 실제 날짜 기준으로 강조
- 달력은 항상 `일~토` 7열 형태 유지
